About Me :

A front-end web/web app developer that specializes in creating interactive, user-friendly, and mobile responsive websites to accommodate all types of web users, Implementing semantic markup and SEO Friendly HTML5 which will result in the site attracting more users and thus creating more revenue for your business.

For building websites my tools are Figma, and AdobeXD to create beautiful layouts, HTML5 to structure content and give it semantic meaning to improve browser efficiency and SEO, CSS3 for the user interface according to layout/wireframe, and JavaScript to give the site an extra hand in performing incredible tasks and improving the user's experience of the site. I have acquired enough experience and skills, I can implement all stages of front-end tasks.
